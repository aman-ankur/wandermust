Discovery Conversation Flow

State machine with 4 adaptive phases — LLM drives content within each

1

🎯 Quick Profile (first-time only)

2-3 adaptive questions max. LLM picks the most important gaps to fill based on what it can already infer. Options are smart: "Indian passport → here's what that unlocks" not just "What passport?"

Transition: when LLM has enough to reason about visas + budget + style

2

💬 Discovery Conversation

Fully adaptive Q&A. LLM asks about this trip — timing, interests, constraints, companions. Each question's options are personalized (budget traveler sees different options than luxury). Brief reactive insights between questions. No fixed question count.

Transition: when LLM has enough signal to start suggesting (min 2 questions, LLM decides "ready")

3

🌍 Progressive Narrowing

This is the magic phase. LLM starts revealing destination ideas: "I'm thinking along these lines..." Shows 4-6 rough ideas with brief reasons. User reacts (like/dislike/tell me more). LLM refines based on reactions — drops some, adds new ones, digs deeper on interesting ones. Multiple rounds possible.

Transition: when user signals they're ready to choose (or LLM senses convergence)

4

✨ Final Reveal

3-5 curated suggestions with rich detail: why it matches, budget breakdown, best timing, what makes it special, and a "what you might not expect" insight. Each suggestion tells a mini-story. User picks one → bridge to optimizer.

Transition: user selects → handoff to "Optimize When"

Key Difference from Current

Current: 10 fixed questions → dump suggestions.
New: Adaptive phases where the LLM decides what matters. Phase 3 (Progressive Narrowing) is entirely new — it's where the "well-traveled friend" energy lives. The system thinks out loud, shares destinations early, and refines based on your reactions.
